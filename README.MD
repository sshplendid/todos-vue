# Todo List with Vue.js

이 repository는 Vue.js 학습을 위한 Todo List Application 소스코드입니다.

## 개발환경
  * Node.js v8.9.4
  * NPM v5.6.0

## 사용 방법
  * 터미널 혹은 CLI 환경에서 소스디렉토리 이동 후 `node app` 명령 실행
  * 웹브라우져에서 `http://localhost:8080` 접속
    * `localhost`로 접속해야 네이버 로그인 API 사용 가능

### 화면 구성
  * Todo Application (`/`)
  * Log In (`/login`)
